### Server side

##### To run server

1. Install dependencies if you are running the application at the first time
    ```shell
    npm install
    ```
2. Launch the server
    ```shell
    npm run start
    ```
    or run it in developer mode

    ```shell
    npm run dev
    ```

Server will start at `http://localhost:5000`, you can open it in browser
